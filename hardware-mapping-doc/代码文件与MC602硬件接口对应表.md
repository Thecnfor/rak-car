## 1. MC602 物理接口类型说明

MC602 控制器板上有 **4 类物理接口**，每类用不同丝印标识：

| 板上丝印 | 接口类型 | 用途 | 对应 dev_id |
|:--------:|---------|------|:-----------:|
| **M1~M6** | 直流电机接口 | 驱动直流减速电机（含编码器反馈） | `0x01`(motor4), `0x02`(motor) |
| **S1~S8** | 舵机接口 | 总线舵机 / PWM 舵机控制 | `0x05`(servo_pwm), `0x06`(servo_bus) |
| **P1~P8** | 多功能 IO 口 | 模拟输入 / 数字 IO / 传感器 / LED | `0x07`(sensor), `0x0E`(led_light), `0x10`(dout) |
| **步进电机口** | 步进电机接口 | 驱动两相步进电机（带细分） | `0x11`(stepper) |

> 同一端口号 + 不同 `dev_id` = 不同物理接口（例如 `port_id=2` + `dev_id=0x0E` 是 P2 口的 LED 灯，`port_id=2` + `dev_id=0x10` 是同一 P2 口的数字输出）。

## 2. M 口 — 直流电机

### 2.1 底盘 4 轮驱动电机（M1~M4）

**底盘类型：** 麦克纳姆轮（MecanumChassis）
**配置位置：** `vehicle/driver/cfg_vehicle.yaml`

```
         车头 ↑
     ┌─────────────┐
M2 ──┤    ╱ ╲      ├── M1   右前轮
     │   ╱   ╲     │
     │   ╲    ╱    │
M3 ──┤    ╲ ╱      ├── M4   右后轮
     └─────────────┘
     左后轮           右后轮
```

| 接口 | 物理位置 | 代码中的初始化 | 运动学编号 |
|:----:|---------|---------------|:---------:|
| **M1** | 🟢 右前轮 | `Motors(port_id=1)` → `WheelWrap` | wheel 1 |
| **M2** | 🟢 左前轮 | `Motors(port_id=2)` | wheel 2 |
| **M3** | 🟢 左后轮 | `Motors(port_id=3)` | wheel 3 |
| **M4** | 🟢 右后轮 | `Motors(port_id=4)` | wheel 4 |

**轮子速度正方向定义：** 顺时针转动（从电机轴方向看）为正。

**编码器：** MC602 编码器为 **真实硬件编码值**，一圈 48 脉冲（12 栅格 × 4 倍频），经减速比 (28/11)^4 ≈ 41.98 后输出轴一圈 = 2015.128 脉冲。

| 文件 | 行范围 | 职责 |

| `vehicle/driver/vehicle_base.py` | L181-214 | `MecanumChassis` 运动学正/逆解 |
| `vehicle/driver/vehicle_base.py` | L307-312 | `CarBase.chassis_init()` — 初始化 `WheelWrap` |
| `vehicle/base/controller_wrap.py` | L267-284 | `Motor4` 类 — 四电机同时控制 |

```

  → self.pump.set(1)    → PoutD(2).set(1)      → MC602 P2 → 真空泵吸气
  → self.valve.set(0)   → PoutD(3).set(0)      → MC602 P3 → 电磁阀关（保气）
  → self.hand_servo.set_angle(夹紧角度)
    → ServoPwm(7).set_angle()                   → MC602 S7 → 手爪闭合

MyTask.arm.grap(0)  # 松开
  → self.pump.set(0)    → PoutD(2).set(0)      → MC602 P2 → 真空泵停
  → self.valve.set(1)   → PoutD(3).set(1)      → MC602 P3 → 电磁阀开（放气）
```

### 8.4 弹射动作链路

```
MyTask.eject(area)                             task_func.py:505
  → Ejection.eject(x, vel)                     task_func.py:29
    → pout.set(1)        → PoutD(4).set(1)     → MC602 P4 → 气阀开
    → motor.set_linear() → MotorWrap(5)        → MC602 M5 → 推杆前进
    → step1.set_rad()    → StepperWrap(1)      → MC602 步进1 → 角度切换
    → pout.set(0)        → PoutD(4).set(0)     → MC602 P4 → 气阀关
```

## 9. 配置文件索引

所有硬件端口映射均由以下配置文件定义：

| 配置文件 | 定义内容 | 关键字段 |
|---------|---------|---------|
| `vehicle/driver/cfg_vehicle.yaml` | 底盘类型、轮子端口、PID 参数 | `port_list: [1,2,3,4]` |
| `vehicle/arm/arm_cfg.yaml` | 机械臂电机/舵机/传感器端口 | `motor.id`, `hand.port`, `grap.port_pump` |
| `config_car.yml` | 摄像头编号、IO 端口、速度限制 | `io.key`, `io.light`, `io.*_sensor` |
| `infer_cs/base/infer.yaml` | 推理服务端口配置 | ZMQ 端口 5001~5004（不涉及 MC602） |
